# 🔐 Laser Security Alarm System

## 📌 Project Overview

The Laser Security Alarm System is a simple electronic security project designed to detect unauthorized intrusion. The system uses a laser beam and an LDR (Light Dependent Resistor) to continuously monitor a protected area. When the laser beam is interrupted, the circuit detects the change in light intensity and activates a buzzer and LED as an alarm indication.

This project demonstrates the practical application of sensors and basic electronic components in security systems.

---

## 🛠 Components Used

- Zero PCB (General Purpose PCB)
- Laser Light Module
- LDR (Light Dependent Resistor)
- Transistor
- Buzzer
- LED
- Resistor
- 9V Battery


---

## ⚙️ Working Principle

1. The laser light is continuously focused on the LDR.
2. The LDR receives constant light and maintains the normal state of the circuit.
3. When an object or person interrupts the laser beam, the amount of light falling on the LDR decreases.
4. The transistor detects this change and switches the circuit ON.
5. The buzzer starts sounding and the LED glows, indicating a security breach.
